# Solo Vale Itineraries

A static multi-page website for GitHub Pages.

## Main pages

- `index.html`: homepage with stylised regional map
- `destinations.html`: destination library by region
- `country.html`: dynamic country information page
- `itinerary.html`: custom itinerary request form
- `blog.html`: blog archive
- `consultation.html`: $70 consultation page
- `about.html`: about and contacts

## Publishing

Upload every file and folder in this directory to the publishing folder selected in GitHub Pages. `index.html` must be directly inside that folder.

## Editing country information

Edit `data/countries.js`. Country pages are generated automatically from this file.

## Before launch

Replace `hello@solovale.com` with the correct email address in `itinerary.js`, `consultation.html` and `about.html`.
